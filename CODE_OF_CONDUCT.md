# Code of Conduct

This project adopts the **Contributor Covenant 2.1** as its community standard. The full canonical text is available at:

- https://www.contributor-covenant.org/version/2/1/code_of_conduct/
- Plain-text mirror: https://www.contributor-covenant.org/version/2/1/code_of_conduct.txt

By participating in this project (contributing code, filing issues, opening pull requests, participating in discussions, or operating a deployment of this Worker), you agree to follow the Contributor Covenant 2.1 standard linked above.

---

## Scope

This applies to:

- All project spaces on GitHub (issues, pull requests, discussions, commits, releases).
- Any public representation of the project, including conference talks, blog posts, and social media posts identifying as maintainers or contributors.

---

## Enforcement contact

Reports of violations can be sent privately to:

**alert@lcvmail.com**

This is the same private channel used for security reports (see [SECURITY.md](./SECURITY.md)). Reports will be acknowledged within 48 hours and handled per the Contributor Covenant 2.1 enforcement ladder.

All reporters will have their identity kept confidential.

---

## Enforcement actions

Per Contributor Covenant 2.1, the maintainer reserves the right to apply any of the following actions in response to a violation:

1. **Correction** — private warning + clarification.
2. **Warning** — formal written warning + temporary restriction on participation.
3. **Temporary ban** — time-bounded removal from project spaces.
4. **Permanent ban** — permanent removal.

Decisions are at the maintainer's discretion guided by the Contributor Covenant 2.1 enforcement ladder.
