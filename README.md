# oraculo-financeiro

> Dashboard avançado de finanças. Integrado aos provedores (ex.: SumUp, Mercado Pago) cruzando dados via IA baseada em GenAI local e processamento com Gemini em Edge.

## Ecossistema LCV Workspace

Este repositório implementa o Oráculo Financeiro.

## Arquitetura & Governança

- **Cloudflare**: Implementado utilizando Workers/Pages, KV, D1, e integrações nativas como AI Gateway.
- **Padrões de Qualidade**: Documentação interna.
